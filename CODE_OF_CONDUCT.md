# Code of Conduct

This project follows the Contributor Covenant Code of Conduct, version 2.1.

## Our Pledge

We pledge to make participation in this project a harassment-free experience
for everyone, regardless of age, body size, disability, ethnicity, sex
characteristics, gender identity and expression, level of experience, education,
socio-economic status, nationality, personal appearance, race, religion, or
sexual identity and orientation.

## Expected Behavior

- Be respectful and constructive.
- Accept feedback gracefully.
- Focus criticism on ideas, code, and project outcomes.
- Avoid harassment, threats, insults, and unwelcome personal attention.

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, code, issues,
and other contributions that are not aligned with this Code of Conduct.

Instances of abusive, harassing, or otherwise unacceptable behavior may be
reported privately to the project maintainer.
